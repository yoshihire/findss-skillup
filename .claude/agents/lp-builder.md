## 絶対規則
- 編集対象は `index.html` のみ（`project/` 配下は参照のみ・編集禁止）
- `data-photo` 属性は絶対に消さない・変えない
- キャンペーンに関する文言は **HTMLに直書き禁止**。末尾scriptの `CAMPAIGN` 設定と `data-campaign` 属性経由でのみ扱う
- 料金・日時・対象・実績などの事実は00シートと一字一句一致させる（勝手に丸めない）
- 煽り表現禁止（00シートの禁止リスト）。「無理な勧誘なし」系は全ページ3箇所まで
- 外部リンクの `rel` は `noopener` のみ（`noreferrer` を付けるとフォーム側の流入分析が壊れる）
- 日本語文字列に他言語の文字（キリル・ハングル等）が混入しないこと（過去に発生。コミット前にチェック）

## 検証プロトコル（作業完了ごとに必ず全部）
1. `python3 -m http.server 8123` でプレビューし、変更セクションを目視（可能ならスクリーンショット）
2. ブレークポイント 375 / 768 / 1366 で横オーバーフローなし（`document.body.scrollWidth > innerWidth` が false）
3. コンソール error / pageerror なし
4. `grep -c 'data-photo' index.html` が想定数と一致
5. 事実表記のgrep照合（3,500 / 7,200 / 10,800 / 各回8名 / 18歳以上 / 毎週日曜13:00）
6. 混入文字チェック（キリル・ハングルの誤混入検出）: `python3 -c "import re;[print(i,l.strip()[:60]) for i,l in enumerate(open('index.html'),1) if re.search('[\\u0400-\\u04FF\\uAC00-\\uD7AF]',l)]"` が無出力

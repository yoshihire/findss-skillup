## 1. 大手スイミングスクール4社の分析

### 共通する「勝ちパターン」（＝借りるべき型）

1. **キッズLPは保護者の不安解消マシン**: 安全体制の具体列挙（監視員数・AED研修・入退館管理）、当日の流れの分単位開示、20問超のFAQ、進級の可視化（JSS62段階・セントラル25段階・イトマン30段階）
2. **大人向けは「本人の動機」起点に切り替わる**: ルネサンス「大人だって遅くない」・ペルソナ分岐（コナミ: 健康習慣/初心者/経験者）・年代性別ラベル付き体験談・機能的ベネフィット
3. **料金は本体サイトでは伏せて店舗へ送る**（チェーン構造ゆえ）— 単独教室のFin-Dは**全額明示できること自体が差別化**になる
4. **キャンペーンは常に「入会金（登録料）を0円化」する形式**が業界標準。月謝は下げない
5. CTAはセクション毎に反復配置（イトマンLPで5〜6回）＋「入会を急かさない」旨の明記

### 最重要ベンチマーク: コナミ「少人数制スイミングスクール」

大手が本気で作った「プレミアム少人数」商品のページ構成が、Fin-Dの直接のお手本になる:
- **「元日本代表選手が教えます」を冒頭に掲出。コーチ9名の経歴（五輪メダル・世界新記録）・保有資格・メッセージをフル掲載**
- 料金を明示: **月3回29,700円・体験3,300円**
- 定員とコーチ数の比率（6名に2名等）を数字で明示

→ **Fin-Dは同等の権威性（元日本代表・日本記録保持者）を月3回10,800円で提供する**。コナミ比1/3の価格。この構造はLPで直接比較こそしないが、「コーチの人格化＋実績のフル開示＋料金明示」という型の有効性が業界最大手で実証されている。

### フィンスイムの扱い
- イトマン姉妹ブランド・セントラルに「フィンスイム」クラスが存在するが、**約40クラス中の1行**に埋没。専用LP・専用訴求なし
- → フィンスイミング専門を名乗り、磨かれたLPを持つだけで**カテゴリの第一想起を独占できる**

## 2. CRO（コンバージョン最適化）定石

- 王道構成「FV→悩み共感→解決→ベネフィット→社会的証明→料金→流れ→FAQ→CTA」— 現行LPはこの型に概ね準拠済み（イトマンTTPMの成果）
- **「FVで即決する層」対応**: 実績数値と価格をFV内に前出しする（現行のオファーカードは正解。維持）
- 料金は松竹梅アンカリング＋おすすめバッジ（現行の月2/月3プラン表示は正解）
- **LINE導線はフォーム比2〜3倍のCVRが出やすい**（BtoC低中単価で顕著）→ LINE相談の副CV設計は正しい。将来「LINE完結型予約」（06実装計画・案C）の根拠
- CTAマイクロコピー（「営業電話はありません」等）でクリック直前の不安を除去 — 現行「無理な勧誘なし」は正解
- **法令**: 希少性・限定訴求は景品表示法（有利誤認）とステマ規制（2023-10施行）に留意。**事実に基づく数字のみ使用**（「各回8名」は事実なのでOK。「残りわずか」の演出は根拠なしでは不可）
- スクール系は素材写真より実写真・講師顔出し・生徒の声（顔写真付き）が信頼に直結

## 3. フィンスイミング市場（日本）

- 競技人口: **全国1,200〜1,800人規模**（JUSF登録者数）。業界当事者が「マイナースポーツ」と自認
- 東京圏の体系立った教室は実質2〜3件。**直接競合はREASのみ**（ビギナー1回3,500円・チケット制2,500〜3,000円/回）。**体験3,500円はREASと完全同額**＝「唯一の競合と同水準」であり安売りでも高値でもない
- **池袋エリア: フィンスイミング専業教室ゼロ**（地理的空白）
- 「フィンスイミング 教室 東京」のSERPに磨かれたLPが不在。比較ポータルもこの競技をカバーせず → **質の高い専用LPを作るだけで検索・転換の両面で優位**
- Fin-D自身のブログ「フィンスイミング界を取り巻く4つのハードル」（①道具入手難 ②指導者不足 ③場所がない ④情報不足）が市場構造を言語化済み → **LPの訴求フレームに転用**（4つとも本クラスが解消する）
- 大人の水泳系の価格相場（訪問者の無意識の物差し）: 体験1,800〜6,600円 ／ グループ月謝7,200〜13,365円 ／ パーソナル1回6,000〜20,000円 → Fin-Dの1回換算3,600円は「公共教室より専門的・パーソナルの半額以下」の中間ポジション
- 訴求戦略の結論: **カテゴリ創造型を主軸**（イルカのように泳ぐ・クロールの1.5倍のスピード等の未知性）＋ 価格の妥当性のみ相場感で補強するハイブリッド。競合名指しは不要

## 4. 大人向けの視覚言語（デザイン調査）

- **大人向けスイミングの実サイトは視覚的に未投資**（テキスト中心・コーポレートテンプレ的）→ デザイン投資自体が差別化になる
- ネイビーの扱い: 知性・信頼を出せるが**単独では冷たい**。白の面積を広く＋体温のある中彩度アクセント（テラコッタ/ペールコーラル等）1色で親しみを足すのが定石 → 04デザインシステムの「navy＋coral＋余白」構成はこの定石に合致
- 写真: **彩度低め・自然光・30〜50代の実年齢被写体・満面の笑みより「集中と静かな充実感」** → 05画像仕様の共通規範に反映済み
- タイポ: プレミアム定石は明朝×欧文セリフ混植だが、「シリアスすぎない」制約から不採用（04参照）。ゴシック太ウェイト＋単色ネイビーで品格を出す
- 参考事例: pilates K（モノクロ＋数値化コピー「10回で違いを知り…」）／ LEMITH パーソナルジムLP（悩み共感→方針→Before/After→声→料金の物語構成）／ Equinox（ダーク基調は上質だが「やさしさ」が消えるため直輸入不可）

現行の視覚言語は「かわいい水中ファミリー」: 高彩度多色パレット（ティール・コーラル・イエロー・グリーン）、魚・気泡・キラキラ・サンゴ装飾、クレヨン風黄色下線、破れ紙バナー＋「こんなに充実しててこの価格!?」、イラスト似顔絵アバター。

## 6. 抜け漏れ検証（opus）— 設計が自覚すべき限界と論点

opusによる批判的検証の結論。**この設計書は以下を「未解決のまま前提を明示して」出荷する**。

### 追加調査・オーナー確認が必要（設計では埋められない）
1. **実顧客の声（VOC）ゼロ**: ペルソナ・障壁の記述は競合とFAQ設計からの推定。実際の問い合わせ・申込者の生の迷い/動機の分析が最優先の次アクション
2. **18歳以上・経験者ゲートの戦略検証**: 大手は全社「泳げなくてもOK」で障壁を下げて集客する中、本クラスは障壁を上げている。全国競技人口1,200〜1,800人の市場でさらに絞る判断が、実来場者データに基づくものかオーナーに確認すべき（誤りとは限らない — 少人数8名の品質担保・専門特化とは整合する）
3. **需要サイジング未確認**: 「SERPが空＝参入余地」とは限らず「需要不在」の可能性もある。検索ボリューム・Instagram等の既存フォロワー規模の確認が必要
4. **流入経路の前提**: 広告か・SNSか・検索か・既存Fin-D顧客かでLPの最適形が変わる。現計画の確認が必要
5. **REAS（唯一の直接競合）の詳細分解**が未実施
6. **素材の実在性**: 実写真・コーチ実績・真正な証言が調達できなければ「本物路線」はAI画像に退行し逆効果。撮影・同意取得の見込み確認が必須

### 設計者判断で確定してよいと検証されたもの（本設計書の判断と一致）
- LP構成順序（王道型）／大人向けトーン転換（ただし「楽しさ・非日常」を殺さない）／体験3,500円の単一数字前面化＋会員プランの分離／CTA文言統一と反復／STEP図解2種（申込後の流れ・当日の流れ）／少人数8名の便益化（ただし「残り◯枠」表示は予約システム連携までやらない）／写真AD仕様／モバイル基準

### リスク登録簿（設計・実装時に能動的に回避）
- **大手定石のカーゴカルト**: 4社は多店舗チェーン構造ゆえ「料金非公開・店舗を探す」— 単独拠点のFin-Dは逆に**全額明示が武器**。定石の取捨選択を能動的に
- **公開中の法務・信頼リスク**: 期限切れ「6月限定」表示は誤認・景表法（有利誤認）リスクを現在進行形で抱えている → Phase 0で即時止血

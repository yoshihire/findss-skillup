# 04. デザインシステム — 「大人の水」

> コンセプト: **スポーツブランドの品格 × 街のスクールの温かさ**。
> シリアスに寄せすぎない（原稿の制約）。ただし「子ども向けスクール」には二度と見えないこと。

## 0. クリエイティブガイドライン原典との関係（2026-07-11 発見・要オーナー承認）

原典GL（fin-d-skillup-form リポジトリ内）の規定: 「水色ベースの、やさしくて楽しいスイミングスクール感＋Fin-Dの信頼感・専門性をうっすら」。色比率 = 白/淡水色60-70%・ブルー/アクア20-30%・イエロー/コーラル/ミント5-10%・**ネイビー5%以内**。避ける: 黒/濃紺主体・金の高級感・煽り文句。

**本設計書との整合**: GLの核（水色ベース・やさしい・金の高級感禁止・煽り禁止）は全て維持する。ただし **「ネイビー5%以内」は対象が小5以上だった時期の規定**であり、18歳以上への転換（2026-06-03）を受けて **「ネイビー10〜15%（見出し・数値・帯2本）」への緩和を提案**する。濃紺"主体"にはしない（背景の主役は白と淡水色のまま）。⚠ この逸脱はオーナー承認事項。承認が得られない場合はネイビーを見出し文字色のみ（5%以内）に抑えるB案で実装可能。

## 1. カラー

### 設計原理
- **ネイビーを権威のアンカーに昇格**: 見出し・数値・フッター・Members帯で使い、ページの背骨にする
- **アクセントは1画面1色**: 現行はミント・ピンク・イエロー・ブルーが同時に踊る（=キッズ感の主因）。1ビューポート内のアクセントはコーラル（CTA）またはティール（達成・チェック）のどちらか一方まで
- イエローはキャンペーンバッジ専用に降格。装飾での使用禁止

### トークン（現行からの差分置換）

```css
:root{
  /* 基幹ブルー（現行継承） */
  --blue-900:#0E4E9E;  /* 見出し・強調 */
  --blue-700:#1B6FD8;  /* リンク・アイコン */
  --blue-500:#4DA3F2;  /* 補助 */
  --blue-100:#DCEEFA;  /* 淡背景 */
  --blue-50:#EAF5FC;   /* セクション交互背景 */

  /* 新設: 権威のネイビー */
  --navy-950:#081F3F;  /* フッター・Members帯の背景 */
  --navy-900:#0B2B57;  /* 大見出し・数値 */

  /* アクセント（役割固定） */
  --coral-500:#F26B7E; /* 主CTAのみ */
  --coral-600:#EB5468; /* 主CTA hover */
  --teal-500:#2FA99B;  /* チェック・達成（現行#36C2B2から彩度-10%で大人化） */
  --yellow:#FFD046;    /* キャンペーンバッジ専用 */
  --line-green:#06C755;/* LINEボタン専用（ブランド指定色） */

  /* テキスト（現行継承） */
  --ink-900:#1E2A3A; --ink-700:#3A4A5E; --ink-500:#6B7A8B; --ink-300:#AFB9C4;
}
```

**廃止**: teal-400／blue-600・blue-200（近接色の統合）。ピンク系背景カード（Members 02カード等）全廃。

### 背景リズム
白 → blue-50 → 白 → … の交互を基本に、**ネイビー帯を2箇所だけ**（実績ストリップ or コーチ／Members）挿入して緩急をつくる。パステル多色帯（現行Membersのミント/ピンク/ブルー/イエロー4色カード）は禁止。

## 2. タイポグラフィ

- 和文: **Noto Sans JP**（継承）／ 欧文・数字: **Poppins**(継承)
- 見出しウェイト 700→**800**（大人向けの確信感。角丸フォントには逃げない）
- 【検討済みの代替案】調査では「見出しに明朝体（和欧混植セリフ）」がプレミアム演出の定石として挙がった（01_調査参照）。ただし本ブランドは「やさしいのに専門的・シリアスすぎない」が制約であり、明朝はシリアス側に振れすぎるため**不採用**。ゴシック太ウェイト＋ネイビー単色で品格を出す。月謝3万円級のプレミアム路線に転換する場合のみ再検討
- 見出し色: グラデーション文字（現行の一部見出し）→ **単色 navy-900** に統一。波下線・勢い線などの見出し装飾は全廃し、eyebrow（小ラベル）＋見出しの2段構成に
- サイズスケール（デスクトップ）: FV見出し 44〜52px ／ セクション見出し 32〜36px ／ カードタイトル 18〜20px ／ 本文 15〜16px・行間1.9
- `word-break:auto-phrase`・`text-wrap:balance/pretty`（実装済み）は継承
- 数字表現: 価格・実績数値は Poppins 700、単位は和文小サイズ（現行の作法を継承）

## 3. 形状・装飾

| 要素 | 現行 | 新 |
|---|---|---|
| カード角丸 | 14〜28px 混在 | **12px 統一**（バッジ・チップのみ999px） |
| セクション区切り | 大波曲線・複数種 | **静かな1種の波カーブ**（振幅小）を上下反転で使い回し |
| 魚・サンゴ・スパークルSVG | 多数浮遊 | **全廃**。泡は「細線の円3個組」1種のみ、1セクション最大1箇所 |

## 4. 写真アートディレクション（全スロット共通規範）

1. **被写体は全員18歳以上**（20代後半〜50代を中心に、ペルソナと同年代）
2. 実在の屋内プール環境。自然なレーンロープ・水面反射。過度な青加工をしない
3. 表情: 「楽しい」より **「集中している・手応えを感じている・和やか」**
4. 文字・ロゴ・バッジの焼き込み一切禁止（文言はHTML側で載せる）
5. フィン種別を文脈と一致（ビ=ビーフィン/モ=モノフィン。05仕様書の指定に従う）
6. トーン統一: 昼光・ややウォーム・彩度控えめ。FVの実写真（P01）を基準トーンとする
7. 顔が判別できる被写体は「本人写真に将来差し替える」前提で、構図をP番号ごとに固定

## 5. コンポーネント規定

- **主CTA**: コーラル・ピル・アイコン付き「初回体験を予約する」。1ビューポート1個まで
- **副CTA**: LINE緑・アウトライン外し単色。常に主CTAとペア配置
- **事実バー/チップ**: 白カード・アイコン先頭・12px角丸。文中でのフォントサイズ切替をやめ、数値のみ強調
- **キャンペーンバッジ**: yellow地・navy文字。campaign設定が active のときだけ描画（06参照）
- **実績数値**: Poppins・navy-900・カウントアップ継承。単位と補足文を必ず添える
- **カード**: 白地・12px・統一影。色地カードは blue-50 のみ許可
- **FAQアコーディオン**: 継承（キーボード操作対応済み）。カテゴリアイコンはネイビー単色線画に統一
- **モバイル固定CTA**: 高さ56px・体験（コーラル）＋LINE（緑）2分割。FV通過後に表示

## 6. モーション

- scroll-reveal（継承）: opacity+8px上昇・400ms・1セクション1回
- カウントアップ（継承）: 実績のみ
- 装飾の浮遊アニメ: **全廃**（reduced-motion対応コードは残す）
- hover: カード浮上2px・ボタン明度変化のみ

## 7. アクセシビリティ・品質基準（継承+強化）

- コントラスト: 本文 4.5:1 以上（blue-500 の白地テキスト使用は禁止 — 現行違反箇所あり）
- `prefers-reduced-motion` 尊重（実装済み）
- 全imgに意味のあるalt＋`data-photo`維持・`loading="lazy" decoding="async"`維持
- 横オーバーフローゼロ（375/390/768/900/1366/1440で検証 — 既存の検証プロトコル継承）

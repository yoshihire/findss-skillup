## 2. 導入・共感（圧縮＋「4つのハードル」フレーム）

- 見出し: 「正しく教わるだけで、泳ぎは変わる。」
  - 現行「100倍楽しくなる」はキッズ文脈の誇張が残るため、大人向けに事実トーンへ（原稿の主旨は維持）
- 本文: フィンの使い方の小さなズレ → 進まない・疲れる → 見てもらえば変わる、の3拍のみ。**現行の長文を約半分へ**
- **追加要素「4つのハードル」ミニストリップ**（市場調査に基づく新規・Fin-D自身のブログフレームを転用）:
  「フィンスイミングを始めるのが難しかった理由 → ここなら全部いりません」

## コピーの全体ルール

1. 一文は短く。見出しはやわらかく、事実は正確に（00シート準拠）
2. 「初めてでも安心」を主語にしない。「見てもらえる」「整える」「取り戻す」を主語に
3. 煽り禁止リスト（00シート）を遵守。緊急性はキャンペーン期限の事実のみで表現
4. 数字は必ず文脈とセットで（「10,000人」単体ではなく「延べ10,000人を指導してきた中で見つけた共通のズレ」のように意味づけ）

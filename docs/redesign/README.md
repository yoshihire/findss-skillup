## この設計書の結論（1分で読む版)

1. **問題の本質**: 対象が「18歳以上の水泳経験者」に確定したのに、ページの視覚言語・写真・世界観が子ども向けスクールのまま。コピーだけ大人向けになっている
2. **戦略**: 大手キッズLPの「型」（導線・不安つぶし・料金の見せ方）は借り、**「見た目」は借りない**。ビジュアルは「大人の水」（ネイビー基調・装飾削減・写真主導）へ転換
3. **市場の追い風**: 池袋にフィン専業教室ゼロ・直接競合は実質1社・大手のフィン講座は埋没 → **専門特化LPを磨くだけでカテゴリ第一想起を独占できる**
4. **最大の追加**: コーチ紹介セクション新設（大人が技術サービスを選ぶ基準は「誰が教えるか」。コナミのプレミアム商品が実証済みの型）
5. **最大の修理**: 期限切れ「6月限定」キャンペーン → 一元管理機構で構造的に再発防止／子ども写真・文字焼き込み画像の全数差し替え（生成プロンプト完備）

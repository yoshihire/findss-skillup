## §2. キャンペーン一元管理機構（仕様）

**目的**: 「期限切れキャンペーンの放置」というバグを構造的に不可能にする。

```html
<script>
/* ===== キャンペーン設定（編集はここだけ） ===== */
const CAMPAIGN = {
  active: true,                    // false で全キャンペーン表示が消える
  label: '7月限定',                // バッジ文言（例）
  benefit: '入会金・事務手数料 0円',
  deadline: '2026-07-31',          // JSTでこの日を過ぎると active でも自動非表示
};
</script>
```

- HTML側: キャンペーンに触れる全要素（FVオファーカードの行・料金カードのバッジ・打消し価格・訴求文）に `data-campaign` 属性を付与
- 描画JS: `active && today <= deadline` のときのみ表示し、`[data-campaign-label]` へ label を注入。それ以外は `display:none` ＋ 料金カードは「入会金・事務手数料 5,000円」の通常表示に自動フォールバック
- **FAQ・本文にキャンペーンの直書きを禁止**（機構外の言及はレビューで却下）
- 現時点の設定値はオーナー判断待ち（延長するなら label/deadline を更新、終了なら active:false）
